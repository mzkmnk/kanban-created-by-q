# Kanban Project Management Tool

Next.js 15 + TypeScript + Tailwind CSSで構築されたモダンなKanbanボード型プロジェクト管理ツール

## 🎯 プロジェクト概要

このプロジェクトは、Trello、Asana、Jiraのようなカード型タスク管理システムを提供するWebアプリケーションです。直感的なドラッグ&ドロップ操作でタスクを管理でき、レスポンシブデザインによりデスクトップ・モバイル両方で快適に利用できます。

## ✨ 主な機能

### 📋 Kanbanボード
- **3カラム構成**: To Do、In Progress、Doneの標準的なワークフロー
- **ドラッグ&ドロップ**: カードを直感的に移動
- **リアルタイム更新**: 変更は即座にローカルストレージに保存

### 🎴 カード管理
- **CRUD操作**: カードの作成・編集・削除
- **優先度設定**: 高・中・低の3段階
- **タグ機能**: カテゴリ分けとフィルタリング
- **タイムスタンプ**: 作成日・更新日の自動記録

### 🎨 UI/UX
- **レスポンシブデザイン**: モバイル・タブレット・デスクトップ対応
- **モーダルインターフェース**: 直感的なカード編集
- **視覚的フィードバック**: ドラッグ中のアニメーション
- **エラーハンドリング**: 適切なエラー表示と回復機能

## 🎮 使用方法

### 基本操作
1. **カードの追加**: 各カラムの「+」ボタンまたは「カードを追加」ボタンをクリック
2. **カードの編集**: カードの編集アイコンをクリック
3. **カードの削除**: カードの削除アイコンをクリック
4. **カードの移動**: カードをドラッグして別のカラムにドロップ

### カード情報
- **タイトル**: カードの見出し（必須）
- **説明**: 詳細な説明（必須）
- **優先度**: 高（赤）・中（黄）・低（緑）
- **タグ**: カテゴリ分けのためのラベル

### データの永続化
- すべての変更は自動的にローカルストレージに保存
- ブラウザを再起動してもデータは保持
- 初回起動時はサンプルデータが自動生成

## 🎨 カスタマイズ

### カラムの追加
`src/utils/storage.ts`の`generateInitialBoard`関数でカラム構成を変更できます。

### スタイルの変更
`src/app/globals.css`でKanban専用スタイルをカスタマイズできます。

### データストレージの変更
現在はローカルストレージを使用していますが、`src/utils/storage.ts`を修正することで外部API連携も可能です。

## 🐛 トラブルシューティング

### よくある問題

**Q: カードが表示されない**
A: ブラウザの開発者ツールでローカルストレージを確認し、必要に応じてクリアしてください。

**Q: ドラッグ&ドロップが動作しない**
A: ブラウザがタッチイベントをサポートしているか確認してください。

**Q: モーダルが表示されない**
A: JavaScriptが有効になっているか確認してください。

### デバッグ方法
```bash
# 開発モードでの詳細ログ
npm run dev

# ブラウザの開発者ツールでコンソールを確認
# ローカルストレージの内容確認: Application > Local Storage
```
